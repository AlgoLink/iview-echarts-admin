# iview-echarts-admin

基于iview-admin和Echarts的一套企业大数据可视化管理后台模板。可用于企业内部管理、业务大数据可视化管理分析。

## Install

``
$ node server.js //启动node端接口服务
``

``
$ npm run init //编译静态文件
``

``
$ npm run dev //启动本地项目
``
